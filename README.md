# AS Innotech Solutions — Website

**Repository:** `asinnotech-website`  
**Description:** Marketing & portfolio site for AS Innotech Solutions — built with Next.js, TailwindCSS, Framer Motion, and Lucide icons. Showcases services, projects, leadership, and contact form connected to the backend API.

## Features

- Modern responsive UI using **Next.js App Router** (`app/`) and **Tailwind CSS**.
- Smooth animations and micro-interactions via **Framer Motion**.
- Iconography with **lucide-react**.
- Pages: Home, About, Services, Portfolio, Contact.
- Contact form that posts to `/api/contact` (API endpoint expected to send/store messages).
- Sections: Hero, Services, Portfolio, Leadership, Trusted partners, Industries, Footer.
- Accessibility-minded components (buttons, form labels, keyboard focus states).

---

## Tech stack

- Framework: **Next.js** (App Router)
- Styling: **Tailwind CSS**
- Animations: **Framer Motion**
- Icons: **lucide-react**
- Dev tooling: Node.js, npm / pnpm / yarn
- Optional backend for contact: any (Node/Express, Cloud Function, or serverless endpoint). Project expects `/api/contact`.
